# Contributing

Thanks for your interest in contributing to Onset!

## Getting Started

1. Fork and clone the repo
2. Install dependencies: `sudo pacman -S gtk4 libadwaita rust` (Arch) or equivalent
3. Build: `cargo build`
4. Run tests: `cargo test`

## Pull Requests

- Open a PR against `main`. CI must pass.
- Keep changes focused. One fix or feature per PR.
- Run `cargo fmt` and `cargo clippy` before committing.

## Reporting Issues

Open an issue on GitHub. Include steps to reproduce if it's a bug.
